## City-Bike-Dev-Academy-task

## Installation

1- Clone the repo 
```bash
https://github.com/capo33/City-Bike-Dev-Academy-frontend
```
2- install dependencies 
```bash
yarn 
```

## Usage
1- Go to the root  directory and start the server
```
yarn start
```
2- Make sure the back end is running
